# Peafowl Calculator
Peafowl Calculator is a tool that helps determine the results of various peafowl breeding pairs, and helps to visualize what specific peafowl will look like

## Versions

### JavaScript Version (New)
Open `index.html` in your browser to use the standalone JavaScript version. This version:
- Runs entirely in the browser
- No server or Python required
- Fast and responsive

### Jupyter Notebook Version
[View the Peafowl Calculator](https://qwertynerd97.github.io/peafowl-calculator/)

# Technical Specs

## JavaScript Version
- `index.html` - Main UI page
- `app.js` - Application logic and data definitions
- `genes.js` - List of all known genes, and how they convert to phenotypes
- `converters.js` - Genotype/phenotype conversion functions
- `children.js` - Calculates the children from the selected parents
- `content/Images/` - Peafowl images
